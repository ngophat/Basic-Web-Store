# Basic Web Store 

This project provides an example about creating web application based on MVC pattern.

## Prerequisites

What things you need to install the software and how to install them

```
Eclipse
MySql database
Glassfish 4.1.2
```
## Built With

* MySql database
* Java Persistence API
* JSP/Servlet
* Boostrap CSS
